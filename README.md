
# Chat System

This is a Socket based chat system that allows you to chat with ur friends by connecting on the same IP address 

## Features

- Real-time chatting 
- no delay
- Fullscreen mode
- Cross platform

## Installation

### Windows 

install From Releases and make sure to have visual studio installed

### Linux

```sh
 git clone https://github.com/for-tristan/Chat-system-project
```
    
## FAQ

#### Can i open 2 chats on the same device?

Yes, by connecting to the local IP address on the two chat windows.

#### Can i still chat if the devices are connected to different networks?

Yes, This project allows u to connect to people even on different networks.
